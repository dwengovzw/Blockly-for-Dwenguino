Class: DwenguinoScenarioUtils

DwenguinoScenarioUtils(scenario, eventBus)

This class is responsible to download or upload the current social robot scenario components configuration to disk as an XML file.

Constructor

new DwenguinoScenarioUtils(scenario, eventBus)

Parameters:
Name Type Description
scenario Object The social robot scenario from which the XML file is composed or uploaded to.
eventBus Object The eventBus that can be used to monitor certain events in the simulator.

Methods

(async) loadScenario()

This function loads the XML file selected by the client and converts it to social robot components in the social robot scenario. It also dispatches a save event to save the current scenario state to local storage.
