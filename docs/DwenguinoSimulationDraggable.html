Class: DwenguinoSimulationDraggable

DwenguinoSimulationDraggable(socialRobotScenario, eventBus)

This class is responsible to make the social robot components in the scenario draggable within their container.

Constructor

new DwenguinoSimulationDraggable(socialRobotScenario, eventBus)

Parameters:
Name Type Description
socialRobotScenario Object The social robot scenario in which the components become draggable.
